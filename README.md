# Bank Management System

A console-based bank management application built with Python and MySQL.

## Features
- User sign-up and login with account authentication
- Deposit, withdrawal, and fund transfer between accounts
- Balance enquiry and transaction history
- MySQL database for persistent data storage
- Unique account number generation for each user

## Tech Stack
- Python
- MySQL
- mysql-connector-python

## How to Run
1. Install the required library:
   pip install mysql-connector-python
2. Run database.py once to set up the tables
3. Run main.py to start the application
